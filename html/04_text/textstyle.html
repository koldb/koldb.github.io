Text Style 텍스트 태그들

b tag : 문자를 두껍게 표시 : bold의 약자

strong tag : 문자를 두껍게, 강조하는 의미

i tag : 문자를 이탤릭체로 표시 : italic의 약자

em tag : 문자를 이탤릭체로 표시 : emphasis 강조하는 의미

u tag : 문자에 밑줄을 표시

s tag : 문자에 가운데 줄(취소선)을 표시

sup tag : 문자를 위첨자로 표시 : super script

sub tag : 문자를 아래첨자로 표시

big tag : 문자를 p태그(기본크기 : 18) 보다 크게 표시

small tag : 문자를 p태그(기본크기 : 18) 보다 작게 표시
